Module log

A module for logging output to STDOUT

Info:

  • Author: mindfilleter

Fields

P8LIBLOGLVL P8LIBLOGLVL configures the logging functions to only output certain log levels.
P8LIBLOGUTC P8LIBLOGUTC configures the output to either use UTC or local time.

Local Functions

log (prefix, any) Log the passed in object to STDOUT The passed in object will be converted to a string and then logged with a timestamp.

Fields

P8LIBLOGLVL
P8LIBLOGLVL configures the logging functions to only output certain log levels. - 0 - disable logging - 1 - errors only - 2 - warnings and errors - 3 - all logs (info, warnings, and errors)
P8LIBLOGUTC
P8LIBLOGUTC configures the output to either use UTC or local time. false (default) for local time, true for UTC
