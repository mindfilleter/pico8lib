Functions

tblrand (tbl) selects a random value from tbl
shuffle (t) randomize the order of a table using fisher-yates shuffle
sort_bubble (t) bubble sort an array in place
sort_bubble_func (t, keyfn) bubble sort an array in place second parameter extracts or calculates a sort key from each item
sort_bubble_func_unsafe (t, keyfn) bubble sort an array in place second parameter extracts or calculates a sort key from each item second parameter is called multiple times per item
sort_quick (t[, a[, b]]) modified quick sort an array in place
filter (t, check) filter a table, keeping only key,value pairs that pass a check function return a new table with the same keys/indices kept from the original
filter_array (t, check) filter an array, keeping only values that pass check function return a new array with sequential indices
any (t) determine if any value in a table is truthy
any_func (t, check) determine if any value in a table passes check function
all (t) determine if every value in a table is truthy
all_func (t, check) determine if every value in a table is truthy or passes check function
all_reverse (t) iterate an array in reverse used just like all()
