Module tables

Get value from table by key, with default

Functions

unshift (t) delete and return the first item from table t
unshift (t, v) add value v to the beginning of table t
insert (t, i, v) insert value v to table t at index i
tblrand (tbl) selects a random value from tbl
shuffle (t) randomize the order of a table using fisher-yates shuffle
sort (t, keyfn) sort an array in place second parameter optionally extracts or calculates a sort key from each item todo test
filter (t, check) filter a table, keeping only values that meet certain criteria return a new table with the same keys/indices kept from the original
filter_array (t, check) filter an array, keeping only values that meet certain criteria return a new array with sequential indices
any (t, check) determine if any value in a table is truthy or passes check function
all (t, check) determine if every value in a table is truthy or passes check function
all_reverse (t) iterate an array in reverse used just like all()

Local Functions

add_f (t, v) add value v to end of table t supposedly faster but less safe than add(t,v)
del_r (t, v) delete value v from table t, do not preserve order
del_i (t, i) delete index i from table t, preserve order
del_ir (t, i) delete index i from table t, do not preserve order
pop (t) delete and return the last item from table t
merge (dst, src) copy keys and values from src to dst also copy metatable if present in src
copy (o) creates a new table copied from o, using merge()
concat (dst, src) adds values from src to end of dst, in order
